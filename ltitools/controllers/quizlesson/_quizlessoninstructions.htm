Quizlesson Component

  • Create a Quiz with 2-5 questions for a topic.
  • Create a Page and write content for a topic.
  • Select Quizlesson from the Rich Editor dropdown.
  • Choose the Quiz you created for this topic.

Now select a question you want to insert into the page. Add more text and questions to the page until all questions are used. When all questions have been added, you will add the Submit Quiz button.
